## 4. 检索结果

测试集结果：

| 配置 | 方法角色 | Doc Recall@5 | Answer Span Hit@5 | Answer Span Coverage@5 |
| --- | --- | ---: | ---: | ---: |
| b1-public-bm25 | B1 | 0.591888 | 0.714286 | 0.591855 |
| b2-public-dense | B2 | 0.510768 | 0.634243 | 0.510746 |
| b3-public-hybrid | B3 | 0.589375 | 0.711773 | 0.589346 |
| b4-public-hybrid-rerank | B4 | 0.633166 | 0.751615 | 0.633144 |
| p-public-hybrid-rerank | P | 0.636037 | 0.756281 | 0.636037 |
| p-public-no-parent | P-no-parent | 0.636037 | 0.628500 | 0.596312 |
| p-public-no-reranker | P | 0.584709 | 0.712491 | 0.584709 |

仅从检索层观察：

- `P` 在测试集 Doc Recall@5 和 Answer Span Hit@5 上略高于 `B4`。
- `P` 与 `P-no-parent` 共享 child ranking，因此 Doc Recall@5 相同；`P` 通过
  parent context 扩展证据，提高 answer span coverage。
- 这些只是检索和证据覆盖观察，不是答案级显著性结论。

自动成功门控：

```text

```

强主张 `P > B4` 不成立，因为 `P-B4` 的 Char F1 bootstrap 95% 置信区间
下界低于 0。Parent context 消融主张成立，因为 `P > P-no-parent` 在 Char F1
和 citation recall 上都有正向 95% 下界。

最终 answer-run manifest 已冻结，包含自动指标、paired bootstrap、success gate
和人工审核聚合结果。

## 8. 论文表述建议

当前安全表述：

```text
在公开 TCM-QG 上，Parent context 能改善 child-only 证据扩展和回答可追溯性；
但在当前正式测试下，完整 Parent-Child 方法尚未表现出相对 B4 hybrid-rerank
基线的稳定答案级优势。
```

不要写 Parent-Child 在答案级显著优于 B4。人工审核可以补充证据支持度和安全性
讨论，但不能在没有额外证据的情况下把自动 `P-B4` bootstrap 结果改写成强成功。

## 9. 局限

- 自动 citation 指标使用“参考答案片段是否出现在被引用公开证据中”来判断；人工审核在
  抽样子集上补充了语义支持和安全性标签。
- 公开 TCM-QG 与古籍 Formal-400 论文数据集彼此独立。
- 大体量检索明细和生成答案产物保留在本地 `data/` 目录，不提交到 Git。
